En este programa lo que hice fue calcular el centroide de varios puntos en el plano. Básicamente, el usuario me dice cuántos puntos va a ingresar y luego voy pidiendo las coordenadas (x, y) de cada uno.

Guardo esos puntos en un arreglo usando una estructura Point, y después llamo a la función calcularCentroide, donde lo que hago es sumar todas las x por un lado y todas las y por otro. Al final divido cada suma entre la cantidad de puntos y así obtengo el promedio, que es justamente el centroide.

Finalmente, muestro el resultado en pantalla con las coordenadas calculadas.
